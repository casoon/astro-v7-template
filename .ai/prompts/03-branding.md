# 03 — Brand Strategy Deep Dive

**Input:** Abgeschlossenes Interview (prompts/02-interview.md)  
**Rollen:** Brand Strategist, Creative Director  
**Output:** Vollständige Brand Foundation

---

## Schritt 1: Positioning Statement

Genau dieses Format. Kein eigenes erfinden.

```
Für [konkrete Zielgruppe — Name der Persona, ihre Situation],
die [konkretes Problem — der echte Schmerz, ihre Sprache],
ist [Angebotsname]
[die/der/das] [Kategorie — wie die Zielgruppe selbst diese Art von Lösung nennt],
die/der/das [wichtigster Nutzen — Ergebnis, nicht Feature],
weil [Begründung — warum wir das besser können als andere].

Im Gegensatz zu [stärkster Wettbewerber oder Status-quo-Alternative],
der/die [deren spezifische Schwäche].
```

**Dann die vier Tests:**

```
Test 1 — Austauschbarkeit:
Setze einen Wettbewerber ein. Stimmt der Satz noch?
→ Wenn ja: Schärfer formulieren.

Test 2 — Überprüfbarkeit:
Kann die Persona beweisen, dass der wichtigste Nutzen stimmt?
→ Wenn nein: Beweis liefern oder Formulierung ändern.

Test 3 — Relevanz:
Ist der Nutzen für die Persona bei ihrer Kaufentscheidung wirklich entscheidend?
→ Wenn nein: Anderen Nutzen wählen.

Test 4 — Spezifität:
Erzeugt die Formulierung ein konkretes Bild im Kopf?
→ "Bessere Qualität" = kein Bild. "Einziger Anbieter der X" = konkretes Bild.
```

## Schritt 3: Brand Voice

**Die Stimme in drei Formulierungen:**

```
Denke an den wichtigsten Vorteil des Angebots.

Falsch (Agentur-Deutsch):
→ "[Wie es eine durchschnittliche Agentur schreiben würde]"

Besser (aber noch generisch):
→ "[Näher dran, aber noch nicht dieser Marke]"

Richtig (diese Marke):
→ "[So klingt genau diese Marke — konkret, differenziert, menschlich]"

Die richtige Formulierung ist der Ton-Standard für alle weiteren Texte.
```

**Schnelltest:**
- Lese den "richtigen" Satz laut vor.
- Könnte ein Mensch das wirklich so sagen? → Wenn nicht: nochmal.
- Klingt es wie ein anderer Wettbewerber? → Wenn ja: schärfer.

**Brand Voice Leitplanken:**

```
Wir klingen wie: [Beschreibung — z.B. "ein erfahrener Kollege der direkt redet"]
Wir klingen nicht wie: [Beschreibung — z.B. "eine Marketingabteilung"]

Wir verwenden: [Sprachliche Merkmale — z.B. "Du-Anrede, kurze Sätze, Aktiv"]
Wir vermeiden: [z.B. "Passiv, Nominalstil, leere Superlative"]

Verbotene Wörter für diese Marke:
- [Wort] — weil [Grund]
- [Wort] — weil [Grund]
```

## Schritt 4: E-E-A-T Strategie

Google und die Zielgruppe beurteilen dieselben Signale.

### Experience
```
Welche echten Erfahrungen haben wir mit diesem Thema?
  Eigene Projekte / Case Studies: [...]
  Wie zeigen wir sie konkret?     [...]
  
Vorher/Nachher (wenn relevant):
  Vorher-Situation der Zielgruppe: [...]
  Nachher-Situation:               [...]
  Konkrete Beweisführung:          [...]
```

### Expertise
```
Welche Fachkompetenz können wir zeigen?
  Prozess-Beschreibung:        [...]
  Erklärung von Konzepten:     [...]
  Blog / Artikel / Ressourcen: [...]
  
Wie zeigt sich Expertise in der Sprache?
  Fachbegriffe die die Zielgruppe kennt (verwenden): [...]
  Fachbegriffe die erklärt werden müssen:            [...]
```

### Authoritativeness
```
Welche externen Signale haben wir?
  Presse: [...]
  Auszeichnungen: [...]
  Partner: [...]
  Zertifikate: [...]
  Kooperationen: [...]
```

### Trustworthiness
```
Vertrauenssignale Pflicht:
  [ ] HTTPS
  [ ] Vollständiges Impressum
  [ ] Datenschutzerklärung
  [ ] Echte Adresse (wenn lokal)
  [ ] Echte Kontaktdaten
  
Vertrauenssignale Optional (je nach Kontext):
  [ ] Echte Bewertungen (Name + Kontext, kein "zufriedener Kunde")
  [ ] Teamfotos (echte Fotos, nicht Stock)
  [ ] Klare Preistransparenz (wenn möglich)
  [ ] Garantien (konkret, nicht vage)
```

## Schritt 5: Farbstrategie

**Farbe als Kommunikation, nicht als Dekoration.**

```
Primärfarbe:
  Funktion: [Was kommuniziert sie? Vertrauen / Energie / Präzision / Wärme]
  Zielgruppenfit: [Warum passt diese Farbe zur Persona?]
  OKLCH-Wert: oklch(L C H)
  Kontrast auf Weiß: [x.x:1] ≥ 4.5:1?
  Dark Mode Variante: oklch(L C H)

Sekundärfarbe:
  Funktion: [Unterstützend, Hintergrund, Kontrast]
  OKLCH-Wert: oklch(L C H)
  Dark Mode Variante: oklch(L C H)

Akzentfarbe:
  Funktion: [Aufmerksamkeit, Highlights, Zähler]
  Sparsamkeit: [Nicht mehr als 10% der Fläche]
  OKLCH-Wert: oklch(L C H)
  Dark Mode Variante: oklch(L C H)
```

**Konkurrierende Farbentscheidungen prüfen:**
- Würde Wettbewerber A ähnliche Farben verwenden? Wenn ja: differenzieren.
- Passen die Farben zu den Adjektiven aus Schritt 2?
- Sind alle Kontrastverhältnisse WCAG-AA-konform?

## Schritt 6: Typografie-Entscheidung

**Vier Kriterien für jede Fontauswahl:**

```
1. Charakter
   Was transportiert diese Schrift? Was "sagt" sie?
   Serif: Tradition, Verlässlichkeit, Autorität
   Sans-Serif: Modern, klar, technisch oder warm (je nach Schnitt)
   Display: Persönlichkeit, Einzigartigkeit, Auffälligkeit

2. Zielgruppenfit
   Würde die Persona diese Schrift als vertrauenswürdig empfinden?
   Oder als zu verspielt / zu steif / zu kommerziell?

3. Lesbarkeit
   Wie liest sich diese Schrift in Bodytexten bei 16px?
   x-Höhe (größer = besser bei klein), Laufweite, Strichstärke.

4. Performance
   Variable Font verfügbar? (Eine Datei = alle Gewichte)
   Self-hosted möglich? (GDPR + Performance)
   Google Fonts? (GDPR-Hinweis nötig, preconnect erforderlich)
```

**Typografie-Pairing:**

```
Display + Body Kombinationen nach Wirkung:

Kontrast (klassisch):
  Display: Serif (z.B. Playfair, Lora, DM Serif)
  Body:    Sans-Serif (z.B. Inter, Source Sans, DM Sans)
  Wirkung: Traditionell + Modern, Eleganz + Klarheit

Harmonie (zeitgemäß):
  Display: Sans-Serif Bold
  Body:    Sans-Serif Regular (gleiche Familie, Variable Font)
  Wirkung: Kohärent, modern, fokussiert

Persönlichkeit:
  Display: Handschrift / Display-Font
  Body:    Neutral Sans-Serif
  Wirkung: Warm, einzigartig, persönlich
  Vorsicht: Nur wenn zur Brand Personality passend
```
