# Copywriting — Textstrategie

## Grundprinzip

Texte die verkaufen, informieren nicht. Sie resonieren.

```
❌ "Wir sind ein erfahrenes Team mit über 10 Jahren Erfahrung."
✅ "In 6 Wochen online. Oder wir arbeiten kostenlos weiter."

❌ "Hochwertige Webdesign-Lösungen für Ihr Unternehmen."
✅ "Endlich eine Website, auf die du stolz bist."
```

## Die Zielgruppe kennen

Bevor du einen Text schreibst, beantworte:

```
1. Was weiß diese Person bereits über das Angebot?
   (Bewusstseinsstufe: Problem-bewusst / Lösung-bewusst / Produkt-bewusst / Kaufbereit)

2. Was ist ihr größter Schmerz gerade?

3. Was ist ihr größter Wunsch?

4. Was hindert sie heute daran zu kaufen?
   (Einwände, Ängste, Alternativen)

5. Mit welchen Worten würde sie ihr Problem selbst beschreiben?
   → Diese Worte verwenden, nicht Branchenjargon
```

## Headline-Formeln

### Problem → Lösung
```
"[Problem erkannt] → [Lösung gefunden]"
"Zu wenig Zeit für Marketing? Wir übernehmen das."
```

### Ergebnis-Versprechen
```
"[Konkretes Ergebnis] in [Zeitraum]"
"In 48 Stunden zu einer Website, die Kunden anzieht."
```

### Kontrast
```
"[Was andere tun] — wir machen [das Bessere]"
"Während andere Websites hübsch aussehen, generiert deine Leads."
```

### Direkte Ansprache
```
"[Zielgruppe] aufgepasst: [Versprechen]"
"Für Handwerker, die mit ihrer Website endlich wachsen wollen."
```

### Frage
```
"Wärst du bereit, [Schmerz] zu beenden?"
"Was wäre, wenn deine Website nachts für dich verkauft?"
```

## Hero-Sektion Struktur

```
H1 (Headline) — Das Versprechen
  Maximal 8-10 Wörter. Klar, konkret, auf die Zielgruppe.
  
Subline — Die Klarstellung
  1-2 Sätze. Wer ist das für? Was genau passiert? Warum jetzt?
  
CTA — Die Handlung
  Verb + konkreter Nutzen, max. 5 Wörter.
  ❌ "Jetzt kontaktieren"
  ✅ "Kostenlose Beratung buchen"
  ✅ "Mein Projekt starten"

Optional: Social Proof
  1 kurze Zahl oder ein Satz: "Über 120 Unternehmen vertrauen uns."
```

## Nutzen vor Features

```
Feature:  "Unser CMS hat eine intuitive Benutzeroberfläche."
Nutzen:   "Du pflegst deine Inhalte selbst — ohne uns anrufen zu müssen."

Feature:  "Wir entwickeln responsive Websites."
Nutzen:   "Deine Kunden finden dich auf dem Smartphone — wo die meisten suchen."

Feature:  "SSL-Zertifikat inklusive."
Nutzen:   "Deine Besucher sehen das Schloss. Vertrauen aufgebaut, Absprung verhindert."
```

Formel: **"[Feature] — das bedeutet für dich: [konkreter Nutzen]"**

## Einwände entkräften

Dokumentiere aus Phase 1 die 3 größten Einwände. Platziere die Entkräftung:
- Im FAQ-Abschnitt (explizit)
- In Testimonials (indirekt durch sozialen Beweis)
- In Risikominimierung beim CTA ("Unverbindlich", "Kostenlos", "In 10 Minuten")

```
Einwand:      "Das ist zu teuer."
Entkräftung:  "Durchschnittliche Amortisation: 3 Monate. ROI: 340%."

Einwand:      "Das dauert zu lang."
Entkräftung:  "Live in 4 Wochen. Garantiert — oder kostenlose Verlängerung."

Einwand:      "Ich weiß nicht, ob das für mich funktioniert."
Entkräftung:  "[Testimonial von jemandem in derselben Situation]"
```

## CTA-Texte

```
❌ Schwach:
  Absenden, Senden, Klicken Sie hier, Mehr erfahren

✅ Stark:
  Verb:    Starten, Buchen, Anfragen, Testen, Erhalten
  Nutzen:  + was genau passiert danach
  
Formeln:
  "Mein [Ergebnis] starten"
  "Kostenloses [Angebot] buchen"
  "[Ergebnis] in [Zeitraum] erhalten"
  "[Produkt] 14 Tage kostenlos testen"
  "Jetzt [Ziel] erreichen"
```

## Vertrauenssignale im Text

```
Zahlen:
  "247 Kunden" statt "viele Kunden"
  "Seit 2015" statt "seit Jahren"
  "4,9 / 5 Sterne (128 Bewertungen)" statt "top bewertet"

Spezifität:
  "Hamburg, München und Berlin" statt "in ganz Deutschland"
  "Durchschnittlich 6 Wochen" statt "schnell"

Garantien:
  "30 Tage Geld-zurück" — konkret
  "Kostenlos bis zur Freigabe" — Risiko nehmen
  "Festpreis — keine Überraschungen" — Angst nehmen
```

## Tonalität nach Markentyp

### Professionell & Seriös
```
Klar, direkt, faktenbasiert.
Keine Ausrufezeichen, keine Superlative ohne Beweis.
"Wir entwickeln seit 2010 digitale Lösungen für mittelständische Unternehmen."
```

### Warm & Menschlich
```
Persönlich, empathisch, auf Augenhöhe.
"Du" statt "Sie" (wenn zum Publikum passend).
"Ich weiß, wie frustrierend das ist. Lass uns das gemeinsam lösen."
```

### Bold & Direkt
```
Kurze Sätze. Klare Aussagen. Keine Weichspüler.
"Andere reden. Wir liefern."
"Deine Website funktioniert. Oder wir arbeiten kostenlos weiter."
```

## Deutsche Texte — Häufige Fehler

```
❌ Agentur-Deutsch vermeiden:
  "Wir bieten maßgeschneiderte Lösungen..."
  "Unser ganzheitlicher Ansatz..."
  "Synergien schaffen..."
  "State-of-the-art Technologie..."

✅ Echte Sprache:
  Wie würde ich das einem Freund erklären?
  Was würde ein echter Kunde sagen?

❌ Passiv vermeiden:
  "Websites werden von uns entwickelt."
  → "Wir entwickeln Websites."

❌ Nominalstil vermeiden:
  "Die Durchführung der Implementierung erfolgt..."
  → "Wir implementieren..."
```

## Textlängen

```
Hero Headline:     6-10 Wörter
Subline:           1-2 Sätze, max 25 Wörter
CTA:               2-5 Wörter
Section Headline:  4-8 Wörter
Paragraph:         max 4-5 Sätze (dann neue Absatz oder Liste)
Liste:             3-7 Punkte, jeder Punkt max 1 Satz
```
